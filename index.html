Starcraft 2 Bank Rebuilder

This tool extracts bankfiles from a replay, then serves them up for you to download

If you are interested in learning about how the page was made, check out the GitHub Page

Directions

Replays are typically found in Documents/StarCraft II/Accounts/{###}/{playerID}/Replays

Upload the replay (.sc2replay) for processing

Generated output files will appear below, with the playerID followed by the name of the bankfile 


  

  
    
  
    Once you get your bank, you will want to place it in the specific
    game's folder; Documents/StarCraft II/Accounts/{###}/{playerID}/Banks/{gameID}
    
    To help with this, the gameID will be displayed by this tool.
